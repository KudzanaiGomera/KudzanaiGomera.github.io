Executive Summary

This investigation was triggered by the detection of suspicious AutoRun registry key modifications, suggesting unauthorized software execution or persistence behavior. Further analysis confirmed the activity originated from a direct download initiated by the user outside the organization's sanctioned software deployment process. Upon engagement, the user explained they encountered issues with the version of Notepad++ available in Software Center and Coupa, and opted to install it from the internet. This behavior, while not malicious, is considered a policy violation. The software was flagged as grayware (PUP), and a removal request was submitted. The user was informed of proper procedures for software installation. This alert is classified as Resolved | Bad BAU. (Location: Corp – South Africa)
